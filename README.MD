# Retail Sales Data Insights
Simple Notebook example to extract actionable insights from e-commerce datasets with Pandas and Matplotlib

(output.png)

# Overview
This project performs an basic analysis of monthly and annual retail sales. Using Python, Pandas and Matplotlib, the analysis indentifies seasonal trends, highlight peak performance periods, and vidualizes sales grownth over a period.

# Key Features
- Data Cleaning and transformation: Handled missing monthly data and datetime format issues.
- Visualizations: Created professional bar charts with highlited peak.
- Trend Analysis: Compared year-over-year performance to identify bussiness grownt patterns.
